
# 문서 보안 서비스

## 얼굴 인식과 손동작 인식을 활용한 2차 문서 보안 서비스

## 프로젝트 개요
스마트폰에 저장된 전자 증명서 및 전자 문서의 보편화에 따라, 중요 문서의 보안을 강화하기 위해 얼굴 인식과 손동작 인식을 활용한 2단계 인증 서비스를 제안했습니다. 이 서비스는 스마트폰의 얼굴 인식 기능과 웹캠을 통한 손동작 인식을 결합하여 문서에 대한 보안을 한층 더 강화합니다.
